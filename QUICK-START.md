# QUICK REFERENCE CARD
## Daily Classroom Use

### BEFORE CLASS:
1. Open Terminal
2. `cd Desktop/math-dashboard`
3. `python3 app.py`
4. Write the Network IP on the board for students

### DURING CLASS:
- Students visit: http://[YOUR-IP]:5000
- They see dashboard with all 5 apps
- Click any app to launch it

### AFTER CLASS:
- Press `Ctrl + C` in Terminal to stop server
- Close Terminal
